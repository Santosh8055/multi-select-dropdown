Single select dropdown

Below example demonstrates the single select dropdown. Click on the dropdown and start typing, values filter automatically. Click on any state from the dropdown to select. Selected state will be bound to a variable and will be shown right side of the dropdown Click on the same state from the list to unselect.

When the dropdown is in focus click on the backspace twice to unselected the selected state.

Multi-select dropdown

Below example demonstrates the multi-value dropdown. Click on any state to select the state. Clicking on the selected state de selects the state. Click the cross button to remove the state from selected states.

Click backspace twice to unselect the last selected state.

Data list dropdown

Similar functionality can be achieved by datalist, which only works in modern browsers as it is an HTML5 element. The Datalist element is used to suggest input values to the user, thereby providing an "autocomplete" feature on form elements. This is especially useful for long lists. Datalist works only for the single select dropdown. This is browser dependent, meaning, it works differently for different browser since it is an HTML attribute.
