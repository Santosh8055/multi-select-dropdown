Single select dropdown

Below example demonstates the single select dropdown. Click on the dropdown and start typing, values filter automatically. Click on any state from the dropdown to select. Selected state will be binded to a variable and will be shown right side of the dropdonw Click on the same state from the list to unselect the

When the dropdown is in focus click on the backspace twice to unselected the selected state

Data list dropdown

Similar funtionality can be achived by datalist, which only works in modern browsers as it is HTML5 element. The Datalist element is used to suggest input values to the user, thereby providing an "autocomplete" feature on form elements. This is especially useful for long lists. Datalist works only for single select dropdown
